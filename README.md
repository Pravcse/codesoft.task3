# CODSOFT Task 3 - Tic Tac Toe Game

## Description
Tic-Tac-Toe Game (Player vs Player)

A simple **Player vs Player Tic-Tac-Toe** game developed in **C++**. This console-based application allows two players to compete by taking turns placing **X** and **O** on a 3×3 board. The game automatically checks for a winner or a draw after every move, providing an interactive and enjoyable gameplay experience.

##  Features

*  Player vs Player gameplay
*  3×3 game board
*  Alternate turns between Player X and Player O
*  Input validation for invalid or occupied positions
*  Automatic win detection
*  Draw detection
*  Interactive command-line interface


##  Technologies Used

* **Language:** C++
* **Concepts:** Object-Oriented Programming (OOP), Arrays, Functions, Loops, Conditional Statements
* **Compiler:** GCC (g++), MinGW, or MSVC
* **IDE:** Visual Studio Code, Code::Blocks, Dev-C++, CLion, or Visual Studio

##  How to Play

1. The game starts with an empty **3×3** board.
2. **Player 1** plays as **X** and **Player 2** plays as **O**.
3. Players enter a number from **1–9** to place their symbol.
4. The game checks for a winner after every move.
5. If all cells are filled without a winner, the game ends in a draw.
